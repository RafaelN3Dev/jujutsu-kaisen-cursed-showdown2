# Jujutsu Kaisen: Cursed Showdown (Web)

Um jogo de luta 2D inspirado no universo de Jujutsu Kaisen, desenvolvido para rodar diretamente no navegador e hospedado via GitHub Pages.

## 🕹️ Como Jogar

### Jogador 1 (Gojo/Outros)
- **A / D**: Mover para esquerda/direita
- **W**: Pular
- **1 / 2**: Ataque Fraco / Forte
- **3 / 4**: Especial / Ultimate

### Jogador 2 (Sukuna/Outros)
- **Setas**: Movimentação
- **Seta Cima**: Pular
- **U / I**: Ataque Fraco / Forte
- **O / P**: Especial / Ultimate

## 🛠️ Tecnologias
- HTML5 Canvas
- JavaScript (Vanilla ES6)
- CSS3 (Custom Design System)
- Toji Fushiguro Update (V3.0)

## 🚀 Deploy no GitHub Pages
1. Faça o commit de todos os arquivos usando o **GitHub Desktop**.
2. No repositório no GitHub, vá em **Settings** > **Pages**.
3. Selecione a branch `main` e a pasta `/ (root)`.
4. Salve e aguarde alguns minutos para o link ficar online.
